# Experiments

This folder contains earlier prototypes and experimental scripts that were developed during the initial exploration phase of the project.

These files are preserved for reference but are **not** part of the current production codebase.

| File | Description |
|------|-------------|
| `gemini.py` | Original monolithic entry point using GitHub Models (GPT-4o-mini) |
| `interactive_cane.py` | Alternative architecture using Google Gemini via Vertex AI |
| `smart_talk_ai.py` | Text-only chat prototype using Gemini (no camera) |

To run the current version of the Smart Cane, use `main.py` in the project root.
